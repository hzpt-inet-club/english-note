# 花心的副词

这一次要来了解一下`形容词`的大兄弟`副词`。

<img src="/adverb/1.png">

如果说中文的形容词是「...的」; 那么副词就是「...地」。

在英语中，这 副词 按 构词法大约分为两类：

「
一类是形容词+ly的派生词，
如，recently，politely，carefully等；
」，

<img src="/adverb/2.png">

「
另一类是不+ly的，
（有些词跟它的形容词还长得一模一样
如，late，fast，hard）
」

<img src="/adverb/3.png">

<img src="/adverb/4.png">
